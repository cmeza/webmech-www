My Source Code

The code used to generate this website is available for perusal on GitHub.com. I started this redesign in mid April 2016 and is still currently in progress.

Front End Repo

https://www.github.com/cmeza/webmech-www

Details
It uses AngularJS 1.5 and assets managed with Bower/NPM. All CSS/JS assets are compiled, concatenated, and minified with Grunt.

Back End Repo

https://www.github.com/cmeza/webmech-api

Details
PHP using Slim 2.6 microservices MVC framework for a REST API. External assets managed with Composer, including a CORS middleware for Slim. The resume object was kept simple as a static JSON object. Some of the future features I'd like to add to the site would require a database and I will use MySQL 5.7.
